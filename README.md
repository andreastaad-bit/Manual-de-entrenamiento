# 🥯 Village Bagel - Sistema de Estandarización de Operaciones y Aseguramiento de Calidad (QA & SOP)

![Village Bagel Operational Header](https://images.unsplash.com/photo-1509440159596-0249088772ff?q=80&w=1200&auto=format&fit=crop)

Este repositorio contiene el ecosistema completo de **Documentación Técnica, Estandarización de Procesos (SOP) y Aseguramiento de la Calidad (QA)** para la operación de la panadería artesanal **Village Bagel**. El proyecto transforma manuales operativos tradicionales y densos en una interfaz digital, interactiva e infográfica de alto nivel, diseñada específicamente para mitigar riesgos, acelerar la capacitación de personal y superar auditorías de inocuidad alimentaria.

***Este proyecto forma parte de mi portafolio profesional para demostrar competencias clave en Ingeniería de Calidad (QA), Arquitectura de la Información, Estandarización de Negocios e Internacionalización (i18n).***

---

## 🎯 Objetivos del Proyecto

El rediseño, traducción estructural y validación de esta documentación técnica se rigió por los siguientes estándares de ingeniería:
1. **Preservación Estricta de la Lógica de Negocio:** Validación cruzada para garantizar cero pérdidas de datos. Se respetó el 100% de los gramajes, temperaturas de horneado, tiempos de leudado y matrices de horarios del documento maestro.
2. **Jerarquía e Interfaz Modular:** Estructuración bajo el principio de *"un concepto por sección"* utilizando bloques independientes, listas de verificación (Checklists) interactivas y flujos secuenciales nativos en CSS.
3. **Internacionalización Espejo (i18n):** Desarrollo de un sistema bilingüe (Inglés y Español) idéntico a nivel de arquitectura de código para asegurar la consistencia operativa en equipos multiculturales.
4. **Optimización de Identidad Visual:** Implementación de la paleta de colores oficial de la empresa basada en un **Rojo Corporativo Profundo** (`#b91c1c`), combinada con fondos crema suaves (`#faf8f5`) para reducir la fatiga visual en entornos operativos.

📐 Características Técnicas y Elementos Infográficos
Para romper la densidad del texto tradicional, se diseñaron componentes nativos utilizando HTML5 y CSS3:

Cajas de Alerta Crítica (.rule-box): Módulos de alto contraste visual con bordes gruesos para blindar políticas inviolables (ej. Protocolos de Salmonella para Huevo Crudo y la Ley de Cero Accesorios en Línea).

Diagramas de Flujo Secuenciales: Rutinas visuales nativas en código (Flexbox) para estandarizar procesos como el Lavado de Manos de 20 segundos y los Ciclos de Sanitización.

Mapeo Vertical de Almacenamiento (.storage-fridge-container): Rejilla modular en CSS que replica la estructura tridimensional del refrigerador industrial (NOM-251), ordenando los insumos listos para consumo en la parte superior y materias primas de alto riesgo en la base.

Control Logs (Bitácoras): Tablas operativas optimizadas con filas alternas y encabezados de alta legibilidad para el llenado diario de inventarios, temperaturas y checklists de limpieza.

🛠️ Instrucciones de Despliegue y Exportación a PDF
1. Visualización Local
Descarga o clona este repositorio en tu computadora.

Haz doble clic en cualquiera de los archivos .html. El manual se desplegará de forma interactiva en tu navegador web (Chrome, Edge, Safari).

2. Generación de Manuales Físicos (Exportación Perfecta)
Para convertir los archivos web en manuales PDF de alta resolución listos para imprenta o distribución digital:

Abre el archivo .html en tu navegador.

Presiona Ctrl + P (o Cmd + P en Mac) para abrir el cuadro de impresión.

Elige la opción Guardar como PDF.

[PASO CRÍTICO]: Despliega el menú de Más configuraciones y marca obligatoriamente la casilla "Gráficos de fondo" (Background graphics). Esto asegurará que el PDF retenga los colores corporativos de las tablas, las sombras de las cajas de alerta y los formatos modulares.

👩‍💻 Competencias Demostradas (Para Reclutadores / QA)
Este proyecto evidencia la aplicación práctica de metodologías de calidad de software y de procesos aplicadas al sector empresarial de servicios:

Aseguramiento de Calidad (QA): Diseño de documentación técnica impecable, validación de flujos de trabajo, control de consistencia lógica inter-idiomas y manejo de casos extremos (Código 86 para desabasto y Código 911 para urgencias).

Gestión de Operaciones (SOP): Creación de manuales detallados de transferencia de conocimientos para puestos clave (Panadero, Masa, Cocinero, Preparador, Cajero).

Cumplimiento Normativo: Alineación de estándares de temperatura y control de vectores biológicos con normativas vigentes de salud e higiene alimentaria (NOM-251-SSA1-2009).

Desarrollo Front-End: Uso avanzado de arquitectura web limpia, selectores CSS jerárquicos y optimización de hojas de estilo orientadas a impresión.
